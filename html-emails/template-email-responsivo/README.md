# Template de E-mail Responsivo — Salesforce Marketing Cloud

Template HTML desenvolvido para campanhas de CRM e Lifecycle Marketing, com estrutura modular para uso em plataformas como Salesforce Marketing Cloud e Braze.

## Objetivo

Criar uma estrutura base de e-mail marketing responsivo, reutilizável e adaptável para diferentes campanhas, mercados e jornadas de CRM.

---

## Estrutura do template

- Header
- Banner principal
- Bloco de texto
- CTA
- Banner secundário
- Footer
- Links com UTM
- Personalização dinâmica

---

## Boas práticas aplicadas

- Estrutura em tabelas para maior compatibilidade com clientes de e-mail
- Largura padrão de 600px
- Uso de atributos inline
- Imagens com `display:block`
- Links parametrizados com UTM
- Separação entre conteúdo visual e estrutura técnica
- Compatibilidade com plataformas de automação
