# Data Views — Salesforce Marketing Cloud

Documentação técnica das principais Data Views utilizadas em Salesforce Marketing Cloud para análise de campanhas, entregabilidade, engajamento e automações.

## Data Views utilizadas

- _Sent
- _Open
- _Click
- _Bounce
- _Unsubscribe
- _Subscribers
- _Journey
- _JourneyActivity

---

## Objetivos

- Análise de performance
- Controle de entregabilidade
- Rastreamento de engajamento
- Exclusão de usuários impactados
- Controle de frequência
- Monitoramento de campanhas

---

## Técnicas aplicadas

- JOIN entre Data Views
- Controle de engajamento
- Segmentação comportamental
- Janela de tempo
- Exclusão incremental
- Queries analíticas

---

## Conceitos aplicados

- CRM Analytics
- Engagement Tracking
- Deliverability
- Marketing Automation
- Lifecycle Marketing
- SQL for SFMC
