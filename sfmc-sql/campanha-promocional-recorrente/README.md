# Campanha Promocional Recorrente — Salesforce Marketing Cloud

Estruturas SQL desenvolvidas para campanhas promocionais recorrentes, recuperação e reengajamento utilizando Salesforce Marketing Cloud.

## Objetivo

Construir segmentações inteligentes para campanhas de CRM focadas em recuperação de usuários, controle de impacto e exclusão de clientes já impactados em campanhas anteriores.

---

## Regras de negócio

- Controle de usuários já impactados
- Exclusão incremental
- Controle de frequência de campanhas
- Segmentação por plano
- Segmentação por engajamento
- Exclusão de usuários sem consentimento
- Controle de status no _Subscribers

---

## Técnicas utilizadas

- Data Views
- ROW_NUMBER()
- LEFT JOIN
- NOT EXISTS
- Controle histórico
- Segmentação comportamental
- Janela de engajamento
- Tratamento de base

---

## Data Views utilizadas

- _Sent
- _Open
- _Subscribers

---

## Objetivos da automação

- Reengajamento de usuários
- Recuperação de clientes
- Controle de frequência de impacto
- Governança de campanhas
- Otimização de entregabilidade
